7、表格标签

表格属性：

  1. width 宽度
  2. height 高度
  3. border 边框
  4. bordercolor 边框颜色
  5. bgcolor 背景颜色
  6. align 对齐：right、left、center
  7. cellspacing 单元格与单元格之间的间距
  8. cellpadding 单元格和内容之间的空隙

行属性：（没有width宽度属性）

  1. height 高度
  2. bgcolor 背景颜色
  3. align 文字水平对齐left、right、center
  4. valign 文字垂直对齐top、middle、bottom

单元格属性：

  1. width 宽度（影响一列的宽度）
  2. height 高度（影响一行的高度）
  3. bgcolor 背景颜色
  4. align 文字水平对齐
  5. valign 文字垂直对齐 表格的合并：colspan合并列、rowspan合并行
