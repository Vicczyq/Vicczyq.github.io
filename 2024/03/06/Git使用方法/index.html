Git使用方法
发布于：2024-03-06 21:24:18 访问：

Gitee和GitHub都为 代码托管平台

生成公私钥

xxxxx@xxxxx.xx是你自己的账号邮箱。

查看公钥cat ~/.ssh/id_rsa.pub

将key添加到gitee

打开Gitee->设置->ssh公钥

粘贴生成的公钥到gitee

克隆

找到空白文件夹右键打开“open git bash here”

其中“xx@xxxxxxxxxx:xx/xx”在gitee项目可以找到

关联本地工程到远程仓库

我们如果本地有了工程文件，然后再在gitee上创建仓库。 此时，可在本地库上使用命令 git remote add把它和 gitee 的远程库关联，如下

git remote -v查看远程库信息

git remote rm 库名删除已有的远程库
